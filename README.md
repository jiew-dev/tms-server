# Teacher Management API

## Running Locally

**Prerequisites**

- Node 22.14.0
- Docker (for Postgres)

- Install dependencies

```
npm install
```

- Start postgres (using Docker here)

```
docker compose up -d
```

- Start dev server

```
npm run start:dev
```

You should be able to access API via Swagger [http://localhost:3000/docs](http://localhost:3000/docs)
